```ts
type Duck = {
  꽥꽥: () => void;
  헤엄: () => void;
};

const duck = {} as Duck; // 타입 단언

// 타입 선언
const duck: Duck = {
  꽥꽥() {
    console.log("꽥꽥");
  },
  헤엄() {
    console.log("어푸어푸?");
  },
};
```

- 타입 단언은 타입스크립트가 개발자의 말만 믿고 Duck 타입으로 인식하여 빈 객체임에도 불구하고 에러를 발생시키지 않는다.
- 타입 선언은 객체 프로퍼티를 모두 채우도록 타입스크립트에서 강제하기 때문에 실수를 저지를 위험이 낮다.
